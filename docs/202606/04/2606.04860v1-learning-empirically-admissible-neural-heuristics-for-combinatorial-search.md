## 摘要
寻找诸如魔方、滑动拼图、Lights Out等组合谜题的最优解路径，一直是人工智能领域的经典挑战。A*等启发式搜索算法只有在使用可容许启发式（即永远不会高估真实剩余代价的启发式）时才能保证路径最优性。深度强化学习方法（如DeepCubeA）训练深度神经网络来近似代价-到-目标启发式。然而，标准均方误差训练经常产生高估，违反可容许性并损害解的最优性。在本文中，我们引入了一个通用的框架，用于学习经过验证校准的可容许神经启发式。我们使用低估的可容许贝尔曼算子结合非对称损失函数训练价值网络，以惩罚高估。为了考虑残差神经函数逼近误差，我们提出了一种基于验证打乱计算的事后校准安全偏移量。我们证明，在评估协议下，我们的校准神经启发式没有观察到可容许性违规，并在实践中保持路径最优性，同时与标准分析基线相比，将搜索节点扩展减少了2×2魔方上的83.0%、3×3 Lights Out网格上的19.9%以及8数码问题上的1.9%。

## Abstract
Finding optimal solution paths for combinatorial puzzles like the Rubik's Cube, sliding tile puzzles, and Lights Out remains a classical challenge in artificial intelligence. Heuristic search algorithms, such as A* , guarantee path optimality only when using an admissible heuristic-one that never overestimates the true remaining cost-to-go. Deep reinforcement learning (RL) methods like DeepCubeA train deep neural networks to approximate cost-to-go heuristics. However, standard mean-squared error (MSE) training regularly yields overestimations, violating admissibility and compromising solution optimality. In this paper, we introduce a generalizable framework for learning validation-calibrated admissible neural heuristics. We train a value network using an underestimating Admissible Bellman Operator combined with an Asymmetric Loss function to penalize overestimation. To account for residual neural function approximation errors, we propose a post-hoc calibration safety offset computed over validation scrambles. We demonstrate that our calibrated neural heuristics achieve no observed admissibility violations under the evaluation protocol and preserve path optimality in practice while reducing search node expansions by up to 83.0% on a 2 by 2 Rubik's Cube, 19.9% on a 3 by 3 Lights Out grid, and 1.9% on an 8-Puzzle compared to standard analytical baselines.

## 2. 论文提出的方法论

### 核心思想
通过三个防御层组合来确保学习到的启发式在训练和验证分布上低估真实代价：
1. **低估的可容许贝尔曼算子**：在目标值中减去安全折扣 $\epsilon$，保证理论目标序列始终低估真实最优代价。
2. **非对称损失函数**（Pinball Loss）：对高估施加很大的惩罚系数 $\alpha$（如 100），迫使网络偏向低估。
3. **事后校准安全偏移**：在验证集上计算最大高估误差 $\delta$，并将其从网络预测中减去，得到最终校准启发式。

### 关键技术细节
- **可容许贝尔曼算子**：
  \[
  T_{\text{ad}} V(s) = \max\left( h_0(s),\; \min_{a}[C(s,a)+V(T(s,a))] - \epsilon \right)
  \]
  其中 $h_0$ 是简单可容许基线启发式，$\epsilon>0$ 为安全折扣。
- **非对称 Pinball 损失**：
  \[
  L_{\alpha}(h_\theta(s), y) = 
  \begin{cases}
  (y - h_\theta(s))^2 & \text{if } h_\theta(s) \le y \\
  \alpha \cdot (h_\theta(s) - y)^2 & \text{if } h_\theta(s) > y
  \end{cases}
  \]
  其中 $\alpha \gg 1$ 惩罚高估。
- **校准偏移**：
  \[
  \delta = \max_{s \in D_{\text{val}}} \max(0, h_\theta(s) - d(s))
  \]
  其中 $d(s)$ 为打乱深度（作为真实代价的上界）。校准后启发式为：
  \[
  h_{\text{calib}}(s) = \max(h_0(s), h_\theta(s) - \delta)
  \]
- **算法流程**（Algorithm 1）：使用课程训练（逐步增加打乱深度），从简单基线初始化，通过上述算子生成目标，用非对称损失更新网络，并定期评估求解率以推进课程，最后在验证集上计算 $\delta$。

## 8. 不足与局限

- **CPU 瓶颈**：A* 搜索的 Python 实现和单状态推理较慢，未使用并行 C++ 加速。
- **扩展到更大谜题**：15-Puzzle、完整 3×3 魔方需要 GPU 训练、更大网络和更长课程周期，本文仅做初步浅层实验。
- **启发式保守性**：安全偏移 $\delta$ 是全局最大值，会导致某些状态大幅低估，增加节点扩展（相比未校准但偶尔超估的模型）。
- **深度保守性**：使用打乱深度作为 $h^*$ 上界可能偏大（因为最优路径可能更短），导致偏移偏大。
- **分布假设**：校准仅保证在验证分布上有效，极端分布外状态仍可能违规。论文承认这一点，并建议未来采用成分校准或不确定性估计。
- **未与更多基线对比**：如 Weighted A*、ARA* 等有界次优搜索虽然在实验中有提及，但未作为主要对比对象（仅在讨论中提及）。
