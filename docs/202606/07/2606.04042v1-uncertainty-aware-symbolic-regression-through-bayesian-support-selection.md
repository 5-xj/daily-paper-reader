## 摘要
确定独立筛选与稀疏化算子（SISSO）框架是一种强大的符号回归方法，用于从大规模非线性特征空间中提取紧凑且可解释的描述符。然而，标准SISSO是确定性的：它返回单个描述符和点预测，而不量化描述符选择、回归系数或预测中的不确定性。在此，我们引入一种概率扩展，其中确定独立筛选（SIS）阶段保持确定性以保持可扩展性，而稀疏化算子（SO）阶段被重新表述为对SIS筛选的支持空间上的贝叶斯推理。由此产生的确定性SIS/贝叶斯SO框架为竞争性描述符支持提供了后验概率、特征包含概率、贝叶斯模型平均预测以及预测可信区间，同时在最大后验极限下恢复标准SISSO的确定性SO描述符。应用于$X_2YZ$ Heusler合金磁矩数据集时，该方法在五折交叉验证RMSE上带来适度改进，并且95%预测区间的经验覆盖率接近标称值。更重要的是，后验揭示了竞争性的、物理相关的符号描述符族，这些在确定性分析中会显得人为唯一。这些结果表明，确定性SIS/贝叶斯SO可用作SISSO的不确定性感知诊断扩展：一种在小数据材料回归问题中评估描述符置信度、稳定性和非唯一性的工具。

## Abstract
The Sure Independence Screening and Sparsifying Operator (SISSO) framework is a powerful symbolic-regression method for extracting compact and interpretable descriptors from large nonlinear feature spaces. However, standard SISSO is deterministic: it returns a single descriptor and point prediction, without quantifying uncertainty in descriptor selection, regression coefficients, or predictions. Here we introduce a probabilistic extension in which the sure independence screening (SIS) stage is kept deterministic to preserve scalability, while the sparsifying operator (SO) stage is reformulated as Bayesian inference over the SIS-screened support space. The resulting deterministic-SIS/Bayesian-SO framework yields posterior probabilities for competing descriptor supports, feature-inclusion probabilities, Bayesian-model-averaged predictions, and predictive credible intervals, while recovering the deterministic SO descriptor of standard SISSO in the maximum-a-posteriori limit. Applied to an $X_2YZ$ Heusler-alloy magnetic-moment dataset, the approach gives modest improvements in five-fold cross-validation RMSE and near-nominal empirical coverage of the 95$\%$ predictive intervals. More importantly, the posterior exposes competing, physically related symbolic descriptor families that would appear artificially unique in a deterministic analysis. These results suggest that deterministic-SIS/Bayesian-SO can be used as an uncertainty-aware diagnostic extension of SISSO: a tool for assessing descriptor confidence, stability, and non-uniqueness in small-data materials regression problems.

### 论文提出的方法论

- **核心思想**：保持 SIS 阶段完全确定性以维持可扩展性，仅将 SO 阶段改造为贝叶斯推断，在 SIS 筛选后的特征子空间上对所有候选支持（固定维度 d）计算后验概率。
- **关键技术细节**：
  1. **确定性 SIS**：用标准 SIS 相关性得分筛选出 K 个特征（K ≪ M），得到简化设计矩阵 Φ_I。
  2. **贝叶斯 SO**：
     - 对每个候选支持 γ（|γ|=d），假设高斯线性模型：y = Φ_I(γ)β_γ + ε, ε~N(0,σ²I)，系数先验 β_γ~N(0,τ²I)。
     - 积分出系数得到闭合形式的边际似然 p(y|γ)。
     - 后验支持概率 p(γ|D_I) = p(y|γ)p(γ) / Σ p(y|γ')p(γ')。
     - 使用三个支持先验：（1）均匀先验；（2）相关-多样性先验，奖励高 SIS 得分、惩罚特征间高相关；（3）Slater–Pauling 先验（仅 d=2），弱鼓励一个电子计数主干加一个残差校正坐标的分离结构。
  3. **预测与不确定性量化**：贝叶斯模型平均（BMA）得到后验均值预测和预测方差，进而构造可信区间；特征包含概率（PIP） = Σ_{γ∋k} p(γ|D_I)。
  4. **超参数固定**：σ² 取标准 SISSO 对应维度的 RMSE 平方（后备为目标方差），τ² 为目标方差除以 d。训练时对特征和目标进行均值中心化以包含截距项。
- **算法流程**：输入数据 → 构建符号库 → 确定性 SIS 筛选 → 在筛选空间枚举所有 d 维支持 → 计算边际似然和先验 → 得到后验分布 → 输出 MAP 描述符、PIP、BMA 预测及区间。

### 论文的主要结论与发现

1. **贝叶斯 SO 改善了预测精度**：d=2 时，BMA 平均验证 RMSE 从确定性基线的 0.5609 降至约 0.5314（均匀先验），d=3 时从 0.4515 降至 0.4467。改进主要发生在 fold 3 等困难分割，确定性 SO 易选错描述符而贝叶斯平均更稳健。
2. **预测区间校准良好**：95% 预测区间的经验覆盖率为 0.9549–0.9660，接近标称值，提供可靠的不确定性估计。
3. **揭示描述符非唯一性**：后验显示特征 8 (V_X² N_X/(N_X+N_Y+N_Z)) 几乎被确定选择（PIP ~0.97），而第二个坐标在特征 201 和 203 之间分布（PIP ~0.53 和 0.38），确定性 SISSO 则只返回其中之一。这种非唯一性在材料分析中具有重要诊断价值。
4. **先验影响有限**：均匀、相关-多样性和 Slater–Pauling 先验给出几乎相同的 RMSE 和 PIP 排序，证明后验主要由似然主导。Slater–Pauling 先验可作为一种可选的解释性偏置，而非性能增强手段。
5. **描述符的物理可解释性**：所选特征族反映了 X 位点体积/价电子与 Y/Z 位点 d 电子填充的对比，与 Heusler 合金的电子结构一致，但不等于参数自由的微观定律。

### 优点

- **方法学亮点**：
  - 保留了 SISSO 的确定性 SIS 阶段，避免了在全空间进行贝叶斯推断的计算灾难，实用性强。
  - 将贝叶斯模型平均引入符号回归，提供包含描述符选择不确定性的完整预测分布，而不仅仅是点估计。
  - 三种先验设计（均匀、数据驱动的相关-多样性、物理弱引导的 Slater–Pauling）覆盖了从无偏好到弱物理先验的谱系，展示了框架的灵活性。
  - 后验特征包含概率直接衡量描述符稳定性，有助于识别化学上合理的替代表达。
- **实验设计亮点**：
  - 五折交叉验证严格分割训练/验证，没有数据泄露。
  - 在 d=2 和 d=3 两个维度验证，并分析改进来源（困难 fold 的稳定化）。
  - 通过折叠 MAP 支持的变化论证了确定性优化的脆弱性和贝叶斯平均的优势。
  - 报告了预测区间覆盖率，评价不确定性校准质量。

### 不足与局限

1. **数据集单一、规模小**：仅在 88 个 Heusler 合金样本上测试，未验证在更大或不同类型材料系统上的表现。小数据下贝叶斯枚举可行，但样本和特征数量增加时枚举所有支持（特别是 d 更大）会指数级增长，目前方法未提供近似后验求解策略。
2. **与更多不确定性量化方法缺乏对比**：未与 bootstrapped SISSO、dropout 神经网络或高斯过程回归等对比，无法判断该方法的相对优劣。
3. **先验设计依赖领域知识且主观**：Slater–Pauling 先验需特定物理先验知识，可能不通用。均匀先验虽通用但忽略了特征间结构信息。相关-多样性先验的超参数（λ_SIS, λ_corr）未调优，固定使用，可能不是最优。
4. **σ² 和 τ² 固定为经验值**：不是完全贝叶斯（即没有对噪声方差和系数先验方差进行共轭更新），这可能低估了不确定性，尤其是当这些固定值与数据不匹配时。
5. **SIS 本身确定性的限制**：贝叶斯推断限于 SIS 筛选后的特征，若 SIS 阶段遗漏了真正重要的特征，后验将无法正确反映全空间不确定性。这一问题未被讨论或刻画。
6. **计算扩展性未评估**：论文仅展示了筛选空间较小（K~300-400）的情况。对于更大 K 或更高维度 d，枚举速度会快速恶化，文中未提供近似后验采样方案（如 MCMC 或变分推断），限制了其在实际大规模问题中的应用。
