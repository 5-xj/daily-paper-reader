## 摘要
大型语言模型（LLMs）近期在自动程序修复（APR）领域展现出强大的潜力，尤其是在通过迭代优化生成并改进补丁候选方面。然而，当前最先进的基于LLM的迭代优化APR方法仍难以完全解决若干挑战，包括维持修复假设之间的有用多样性、识别语义相关的修复族、组合互补的部分修复、利用结构化故障信息以及逃离结构缺陷的搜索区域。本文提出一种基于种群的语义进化APR迭代优化框架，称为EvolRepair，将基于LLM的APR形式化为一种语义进化算法。EvolRepair重新定义了经典遗传算法的搜索范式，用由LLM和结构化执行反馈驱动的语义感知组件替代了语法层面的操作符。候选修复被组织为行为一致的群组，使算法能够保持多样性、推理修复族之间的关系，并通过跨种群重组互补修复洞见来综合出更强的候选方案。通过利用结构化故障模式引导搜索方向，EvolRepair既可优化有前景的修复策略，又能在必要时转向替代抽象。实验结果表明，EvolRepair在修复效果上显著优于现有的基于LLM的APR方法。

## Abstract
Large language models (LLMs) have recently shown strong potential for automated program repair (APR), particularly through iterative refinement that generates and improves candidate patches. However, state-of-the-art iterative refinement LLM-based APR approaches cannot fully address challenges, including maintaining useful diversity among repair hypotheses, identifying semantically related repair families, composing complementary partial fixes, exploiting structured failure information, and escaping structurally flawed search regions. In this paper, we propose a Population-Based Semantic Evolution framework for APR iterative refinement, called EvolRepair, that formulates LLM-based APR as a semantic evolutionary algorithm. EvolRepair reformulates the search paradigm of classic genetic algorithm for APR, but replaces its syntax-based operators with semantics-aware components powered by LLMs and structured execution feedback. Candidate repairs are organized into behaviorally coherent groups, enabling the algorithm to preserve diversity, reason over repair families, and synthesize stronger candidates by recombining complementary repair insights across the population. By leveraging structured failure patterns to guide search direction, EvolRepair can both refine promising repair strategies and shift toward alternative abstractions when necessary. Our experiments show that EvolRepair substantially improves repair effectiveness over existing LLM-based APR approaches.

## 一、研究背景与核心问题

- **领域背景**：自动程序修复（Automated Program Repair, APR）旨在通过程序分析、测试和搜索等手段自动生成修复补丁，从而减少人工调试。  
  随着**大型语言模型（LLM）**的发展，APR 转向基于自然语言到代码的生成任务，尤其是**迭代优化式 LLM 修复**在性能上取得显著进展。

- **主要问题**：现有迭代式 LLM-APR 框架（如 ChatRepair、REx）仍存在五大核心缺陷：
  1. 难以保持修复假设的语义多样性；
  2. 无法识别语义相近或互补的修复族；
  3. 无机制合成不同候选间的部分修复；
  4. 对失败信息利用不足，仅依赖通过率；
  5. 搜索过程易陷入局部最优（无法抽象层级转换）。

- **研究目标与意义**：本文希望通过引入**语义层面的种群演化**机制，实现在种群维度上并行探索修复空间，平衡搜索的**多样性与方向性**，提高 LLM 自动修复的可靠性与质量。

## 二、方法论：EvolRepair 框架

### 1. 核心思想

- 将 LLM 驱动的 APR **形式化为语义进化算法（Semantic Evolutionary Algorithm）**。
- 通过**种群层次的搜索**代替单候选迭代，使算法在多条修复轨迹上并行演进。
- 引入**语义感知操作符**（mutation、crossover）和**行为一致群组**形成机制，实现多样化保留与部分修复重组。

### 2. 框架总体结构  

EvolRepair 的算法流程可简述为以下步骤：

1. **初始化（Initialization）**  
   使用 LLM 生成初始修复候选种群，每个候选均通过测试集计算其通过率（Fitness）与行为签名（即通过的测试集合）。
2. **行为聚类（Behavioral Grouping）**  
   基于候选间的测试通过集，计算 *Jaccard 相似度*，按行为相似性分组。
3. **跨组采样（Cross-group Sampling）**  
   抽取多个组间组合，增强多样性，使后续重组能综合不同失败模式的修复特征。
4. **语义重组（Population-level Recombination）**  
   以一个组或混合组为输入，由 LLM 综合多个候选逻辑生成新的融合修复候选。
5. **语义变异（Semantic Mutation）**  
   基于失败样例反馈信息（错误输入、输出、异常信息），用 LLM 定向改进单个候选。
6. **存活选择（Survivor Selection）**  
   删除重复候选，按通过率、测试覆盖、代码长度综合排名，选取前 N 个进入下一代。
7. **终止条件**  
   若出现通过所有测试的正确修复或达到最大代数，则结束搜索。

算法公式中引入：
- **Fitness**：通过率 \( F(P') = \frac{1}{M}\sum I[\text{Eval}(P', t_j) = 1] \)
- **Behavioral Signature**：通过测试集索引集合 \( \psi(P') = \{ j | \text{Eval}(P', t_j) = 1 \} \)
- **相似度**：Jaccard 相似度 \( sim(P_i, P_j) = \frac{|\psi(P_i)\cap \psi(P_j)|}{|\psi(P_i)\cup \psi(P_j)|} \)
