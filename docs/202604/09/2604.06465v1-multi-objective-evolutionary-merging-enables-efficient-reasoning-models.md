## 摘要
推理模型通过利用长链的思维过程在解决复杂问题方面展现出卓越的能力。然而，这种更为深思熟虑的推理在推理阶段带来了大量的计算开销。Long-to-Short (L2S) 推理问题旨在使用更少的 token 保持高精度，但现有的无训练模型融合方法依赖于标量化、固定超参数的算术手段，这些方法具有高度脆弱性，并强迫模型在性能上做出次优折衷。为弥补这一差距，我们提出了 Evo-L2S，这是一种新颖的框架，将 L2S 推理形式化为一个多目标优化挑战。通过进化式模型融合，Evo-L2S 明确优化了准确性与输出长度之间的权衡，以生成一系列稳健的合并模型的 Pareto 前沿。为了使针对大型语言模型的搜索在计算上可行，我们提出了一种基于熵的子集采样技术，极大地降低了适应度估计的开销。我们在 1.5B、7B 和 14B 参数规模的六个数学推理基准上进行了全面实验，结果表明 Evo-L2S 能够在保持甚至提升原始推理模型问题求解准确率的同时，将生成的推理轨迹长度减少超过 50%。

## Abstract
Reasoning models have demonstrated remarkable capabilities in solving complex problems by leveraging long chains of thought. However, this more deliberate reasoning comes with substantial computational overhead at inference time. The Long-to-Short (L2S) reasoning problem seeks to maintain high accuracy using fewer tokens, but current training-free model merging approaches rely on scalarized, fixed-hyperparameter arithmetic methods that are highly brittle and force suboptimal compromises. To address this gap, we introduce Evo-L2S, a novel framework that formulates L2S reasoning as a multi-objective optimization challenge. By leveraging evolutionary model merging, Evo-L2S explicitly optimizes the trade-off between accuracy and output length to produce a robust Pareto front of merged models. To make this search computationally tractable for large language models, we propose an entropy-based subset sampling technique that drastically reduces the overhead of fitness estimation. Comprehensive experiments across 1.5B, 7B, and 14B parameter scales on six mathematical reasoning benchmarks demonstrate that Evo-L2S can reduce the length of generated reasoning traces by over 50% while preserving, or even improving, the problem-solving accuracy of the original reasoning models.

## 一、研究问题与总体背景

- **核心问题**：  
  大规模推理型语言模型（LLMs）在复杂数学与逻辑任务中表现优异，但需生成冗长的链式思维（Chain-of-Thought, CoT）推理过程，导致推理延迟、计算开销大。许多任务其实不需要如此“深思熟虑”的 System 2 推理，从而出现“过度思考”（overthinking）问题。
  
- **研究动机**：  
  现有应对“长到短推理（Long-to-Short, L2S）”任务的无训练融合方法（如 Task Arithmetic、TIES、ACM）存在**超参数敏感性高、可迁移性差、结果脆弱**等问题。模型在压缩推理长度与保持高准确率之间难以取得稳健平衡。

- **研究目标**：  
  在**无需额外训练**的前提下，通过**参数空间中的进化式优化**自动搜索模型融合策略，实现：
  - **推理准确率**与**输出长度**的兼顾；
  - 自动求得稳健的 **Pareto 最优前沿（Pareto front）**。

## 二、方法论：Evo-L2S 框架

### 1. 核心思想
- 将 **L2S 推理任务**形式化为一个 **双目标优化问题**：
  - （1）最大化模型推理准确率；
  - （2）最小化输出长度（即生成的 token 数）。
- 不采用固定超参数，而利用多目标进化算法 **NSGA-II** 自动探索 **两个端点模型（System 1 vs System 2）** 之间的权衡。

### 2. 模型合并策略
- 所有模型共享相同的预训练起点 \( \theta_0 \)，定义两个端点：
  - System 1（快思考、简洁输出）：Qwen2.5-Math；
  - System 2（慢思考、高准确）：DeepSeek-R1-Distill-Qwen。
- 参数位移向量定义为：
  \[
  \tau = \theta_{S1} - \theta_{S2}
  \]
- **Task Arithmetic 融合公式**：  
  \[
  \theta_M = (1 - \lambda)\theta_{S2} + \lambda\theta_{S1}
  \]
  其中 \( \lambda \in [0,1] \) 为进化算法的基因（genotype）。

### 3. 多目标进化优化
- 使用 **NSGA-II 算法** 搜索 Pareto 前沿：  
  每一代通过选择、交叉、变异生成候选模型，依据两个目标的非支配排序（non-dominated sorting）与拥挤距离（crowding distance）保留多样化的最优解。
- **适应度函数**：  
  \[
  F(M) = [-Acc(M), Len(M)]
  \]
  其中：
  - \(Acc(M)\)：在子集上的正确率；
  - \(Len(M)\)：平均输出长度。

### 4. 熵基子集采样（Entropy-based Subset Sampling）
- **问题**：每代需评估数百个模型，完整基准测试代价极高。
- **解决思路**：  
  - 先用若干线性融合（λ 等间距）模型组成“校准池”；
  - 计算每道题被各模型答对的比率 \( p_i \)，并据此计算 **Bernoulli 熵**：
    \[
    H_i = -p_i\log_2p_i - (1-p_i)\log_2(1-p_i)
    \]
  - 选择熵最高的 50 道题作为固定评估子集，用于进化优化的适应度估计。
- **优势**：在仅 50 样本情况下仍能保持高秩序保真度（Spearman ρ ≈ 1.0），大幅降低计算代价。
