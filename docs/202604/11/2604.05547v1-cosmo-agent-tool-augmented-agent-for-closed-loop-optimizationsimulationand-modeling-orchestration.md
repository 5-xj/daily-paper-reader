## 摘要
迭代的工业设计–仿真优化过程受制于 CAD-CAE 语义鸿沟：在多样且耦合的约束条件下，将仿真反馈转化为有效的几何编辑具有挑战性。为弥合这一鸿沟，我们提出 COSMO-Agent（闭环优化、仿真与建模编排），这是一种工具增强的强化学习（RL）框架，用于教会大语言模型（LLM）完成闭环 CAD-CAE 流程。具体而言，我们将 CAD 生成、CAE 求解、结果解析及几何修正建模为一个交互式 RL 环境，使 LLM 能够协同外部工具并持续修订参数化几何直至满足约束。为使学习过程稳定且可工业应用，我们设计了多约束奖励函数，以同时鼓励可行性、工具链鲁棒性和结构化输出的有效性。此外，我们还提供了一个与工业场景对齐的数据集，涵盖 25 种组件类别，并包含可执行的 CAD-CAE 任务，用于支持真实的训练与评估。实验结果表明，COSMO-Agent 的训练显著提升了小型开源 LLM 在基于约束设计方面的能力，在可行性、效率与稳定性等维度上超越了大型开源及强闭源模型。

## Abstract
Iterative industrial design-simulation optimization is bottlenecked by the CAD-CAE semantic gap: translating simulation feedback into valid geometric edits under diverse, coupled constraints. To fill this gap, we propose COSMO-Agent (Closed-loop Optimization, Simulation, and Modeling Orchestration), a tool-augmented reinforcement learning (RL) framework that teaches LLMs to complete the closed-loop CAD-CAE process. Specifically, we cast CAD generation, CAE solving, result parsing, and geometry revision as an interactive RL environment, where an LLM learns to orchestrate external tools and revise parametric geometries until constraints are satisfied. To make this learning stable and industrially usable, we design a multi-constraint reward that jointly encourages feasibility, toolchain robustness, and structured output validity. In addition, we contribute an industry-aligned dataset that covers 25 component categories with executable CAD-CAE tasks to support realistic training and evaluation. Experiments show that COSMO-Agent training substantially improves small open-source LLMs for constraint-driven design, exceeding large open-source and strong closed-source models in feasibility, efficiency, and stability.

## 二、方法论：核心思想与技术细节

### 1. 核心思想
- **提出框架**：COSMO-Agent（Closed-loop Optimization, Simulation, and Modeling Orchestration）是一种 **工具增强型强化学习（RL）智能体**。  
- **关键思路**：将 CAD 生成、CAE 仿真、结果提取与几何参数修正视为一个 **交互式强化学习环境**；LLM 作为策略主体学习如何协调外部工具并迭代设计，直到所有约束满足。

### 2. 系统构成
包括三个主要模块（对应 Fig.2）：
1. **通用框架**：以任务输入（类别、初始参数、材料库和加载条件）驱动多轮设计 → 仿真 → 校正闭环。
2. **MCP 工具集（Multi-tool CAD–CAE Pipeline）**：涵盖  
   - CAD 生成（基于 CadQuery，输出 STEP 模型）；  
   - CAE 求解（调用 FreeCAD FEM + Gmsh + CalculiX）；  
   - 结果提取（计算最大位移和最大等效应力）；  
   - 成本计算（基于体积、密度和价格）。  
3. **奖励设计（Reward Function）**：利用多目标强化学习综合优化鲁棒性与可行性。

### 3. 强化学习与奖励设计
- 使用 **GRPO（Generalized Reinforcement Policy Optimization）** 算法训练，融合三类奖励：
  1. **约束奖励 \(R_{cons}\)**：根据满足的约束数量（位移、应力、成本）给予分段奖励。  
  2. **停止奖励 \(R_{stop}\)**：在首次满足全部约束后继续调用工具的行为给予惩罚，促进效率。  
  3. **结构化输出奖励 \(R_{fmt}\)**：确保最终输出为一致的可解析 JSON，保证可复现。  

总体目标：
\[
R = R_{cons} + R_{stop} + R_{json}
\]

- 这种基于 **rollout 日志的奖励机制** 避免了额外的再仿真成本，直接从工具交互序列中计算反馈。

### 4. 交互与公式流程
- 在每一轮 \(t\)：LLM 根据当前设计状态 \(x_t = (c, p_t, m_t)\) 生成新设计；  
- MCP 工具返回反馈三元组：
  \[
  Φ(x_t; η) = (u_{max}^{(t)}, σ_{max}^{(t)}, C^{(t)})
  \]
- 约束检查：
  \[
  u_{max}^{(t)} ≤ δ, \quad σ_{max}^{(t)} ≤ σ_{allow}(m_t), \quad C^{(t)} ≤ κ
  \]
- 若不满足，则更新参数至 \(p_{t+1}, m_{t+1}\)，否则停止。

这一闭环过程使 LLM 具备“数值反馈驱动的几何优化”能力。

## 三、实验设计与评估策略

### 1. 数据集与场景
- 构建了 **工业组件数据集**，共 **20,000 训练样本 + 200 测试 + 100 泛化样本**。  
- 覆盖 **25 类常见工业部件**（如法兰盘、支撑架、螺母、工字梁等）。  
- 提供统一材料库（5种材料，含弹性模量、泊松比、密度、单价、允许应力）。  
- 任务包含三类约束：**位移、强度、成本**，并设有随机难度等级（约束缩减 5–30%）。

### 2. Benchmark 与对比模型
- **工具链与评估统一**：所有模型均使用相同 CAD–CAE 环境和交互上限。  
- **对比模型**：
  - 开源：Qwen3-8B、Qwen3-30B、Qwen3-Next、Intern-S1-mini、Intern-S1、Llama-4-Scout。  
  - 闭源：Claude-Sonnet-4.5、Gemini-3-Flash。  
- 评价指标：
  - 全成功率（FSR）、位移/应力/成本单项满足率（DSR/SSR/CSR）；  
  - 可解析输出率（MEO）；平均得分（AS）；平均工具调用数（ATC）。

## 六、主要结论与发现

- COSMO-Agent（8B）在闭环CAD–CAE优化中实现：
  - **FSR 74.5%**（领先所有模型）；  
  - **DSR 87.5%、SSR 76.0%、CSR 93.5%**，结构化输出率 100%。  
- 相较最强开源模型（Intern-S1 236B），可行性提升超 40%；相较闭源 Gemini-3-Flash 提升约 7%。  
- 且工具调用更少（ATC 6.72 vs Gemini 9.32），体现 **更高交互效率** 。  
- 在未见类别上 FSR 仍达 75%，显示出较好的跨模板泛化能力。  
- 消融实验表明：  
  - 无RL训练性能降至26%；  
  - roll-out奖励缺失则FSR降至36%，确认奖励设计核心贡献。
