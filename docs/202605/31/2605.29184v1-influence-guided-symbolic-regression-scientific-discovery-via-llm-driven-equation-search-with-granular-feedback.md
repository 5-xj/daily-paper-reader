## 摘要
大语言模型（LLMs）为科学发现提供了有前景的途径，但它们在符号回归中的应用常常受到低效的搜索策略和粗略反馈信号的限制。当前方法通常使用标量指标（如全局均方误差）引导LLMs，这无法识别所提出方程中哪些组成部分驱动了性能或导致了误差。我们引入了影响引导的符号回归（IGSR），该方法将方程发现构建为一个迭代的两步过程，结合多样化的项生成与严格的选择：LLM为线性模型生成候选基函数$ψ_j(\mathbf{x})$，然后使用细粒度影响评分$Δ_j$对其进行评估。这些评分量化了每个项对泛化准确性的边际贡献，从而实现影响引导的剪枝过程，系统地优化模型结构。将该机制集成到蒙特卡洛树搜索（MCTS）中，可以在导航组合搜索空间的同时，平衡对新函数形式的探索与对高影响项的开发。我们在多个基准测试上展示了IGSR的有效性，包括LLM-SRBench、药理学PKPD模型、流行病学模拟和真实基因组数据。值得注意的是，我们在一个高维生物数据集的案例研究中验证了该框架进行真正发现的能力，其中IGSR识别出DNA甲基化与RNA聚合酶II暂停之间的新关系；该假设随后通过湿实验得到了支持。

## Abstract
Large Language Models (LLMs) offer a promising avenue for scientific discovery, yet their application to symbolic regression is often constrained by inefficient search strategies and coarse feedback signals. Current methods typically guide LLMs using scalar metrics (e.g., global Mean Squared Error), which fail to identify which components of a proposed equation are driving performance or causing error. We introduce \textit{Influence-Guided Symbolic Regression} (IGSR), a method that frames equation discovery as an iterative two-step process combining diverse term generation with rigorous selection: an LLM generates candidate basis functions $ψ_j(\mathbf{x})$ for a linear model, which are then evaluated using granular influence scores $Δ_j$. These scores quantify each term's marginal contribution to generalization accuracy, enabling an influence-guided pruning process that systematically refines the model structure. Integrating this mechanism into a Monte Carlo Tree Search (MCTS) enables navigating the combinatorial search space while balancing exploration of novel functional forms with exploitation of high-influence components. We demonstrate IGSR's effectiveness on a diverse suite of benchmarks, including LLM-SRBench, pharmacological PKPD models, an epidemiological simulation, and real-world genomic data. Notably, we validate the framework's capacity for genuine discovery in a case study using a high-dimensional biological dataset, in which IGSR identified a novel relationship between DNA methylation and RNA Polymerase II pausing; a hypothesis that was subsequently supported via wet-lab experimentation.

## 1. 核心问题与整体含义

**研究动机**：大语言模型（LLM）在符号回归中具有潜力，但现有方法仅使用全局标量误差（如均方误差MSE）作为反馈信号，无法识别方程中哪些具体组分对性能有贡献或导致误差。这种粗粒度反馈导致搜索效率低，依赖LLM生成先验而非数据驱动推理。

**核心问题**：如何设计一种结构化、细粒度的反馈机制，使LLM能够高效地发现紧凑、可解释的数学方程，并平衡探索与利用。

**整体含义**：IGSR通过引入逐项影响分数解耦生成与选择，将符号发现从全局优化转向组件级精炼，显著提升搜索效率，并具备真正的科学发现能力（案例中湿实验验证了AI生成的假设）。

## 2. 方法论

### 核心思想
- **目标形式**：发现稀疏线性模型 $f(\mathbf{x}) = \sum_j w_j \psi_j(\mathbf{x})$，其中 $\psi_j$ 是LLM生成的（可能非线性的）基函数，$w_j$ 通过线性回归优化。
- **关键创新**：用**逐项影响分数** $\Delta_j$ 替代全局标量损失，量化每个基函数对验证集泛化精度的边际贡献。以此驱动一个“提出-剪枝”（propose-and-prune）循环，并嵌入蒙特卡洛树搜索（MCTS）进行系统探索。

### 关键技术细节
1. **提出阶段**：LLM“提出”代理根据任务描述、当前项集、历史决策生成约5个候选基函数（首次10个）。
2. **评估与影响计算**：将新项与现有项合并，用OLS拟合线性模型，然后在验证集上计算每个项的影响分数：
   - $\Delta_j = \text{MSE}(\text{full}) - \text{MSE}(\text{without term } j)$，即去掉该项（设 $w_j = 0$）后验证MSE的增加量。
   - 默认使用“不重新拟合”的快速计算方式。
3. **剪枝阶段**：
   - **默认IGSR**：基于 $\Delta_j$ 确定性保留 top-K（K=6）项。
   - **IGSR-Agent**：由第二个LLM代理基于 $\Delta_j$ 和语义推理决定保留/丢弃。
4. **搜索策略**：上述循环嵌入MCTS。每个节点代表一个方程状态；从父节点生成多个子节点（通过LLM随机采样）；使用UCT平衡探索与利用；奖励为负验证MSE；默认使用启发式MCTS（立即奖励回传，不执行完整rollout）。

### 算法流程（文字描述）
1. 初始化MCTS树（根节点为空集合）。
2. 对每个MCTS迭代：选择 → 扩展（执行propose-and-prune循环生成子节点） → 评估（计算验证MSE作为奖励） → 回传。
3. 直至预算（30次扩展）耗尽，返回验证MSE最优的方程。

  - 剪枝鲁棒性（共线性、交互项信号）
  - 跨种子方程稳定性（Jaccard相似性）
  - Term-local Optimization（TLO变体）
  - 提出的项数影响
  - 提示词变体敏感性
  - 搜索中候选项多样性分析
- **公平性**：所有方法限制相同LLM（除非指定）、固定预算（如300k token/run）、相同最大项数（6）。对于不可比的基线（如SINDy）有额外对比。实验设计充分、客观。

## 6. 主要结论与发现

1. **性能优势**：IGSR在6个基准数据集中5个取得白箱方法中最低MSE，平均排名1.17；在LLM-SRBench上预测性能平均排名第一，符号恢复指标也优于基线。
2. **细粒度反馈的价值**：消融实验表明，去除影响分数或MCTS均导致MSE大幅上升（0.00079→0.293→0.626→4.85），且影响分数反馈能加速收敛。
3. **高维可扩展性**：在RNAPol数据集（263特征）上，IGSR运行时随特征数亚线性增长（10→263特征耗时仅增25%），而传统方法（OpenFE、SINDy）因组合爆炸失败或超时。
4. **跨种子稳定性**：在已知真值的数据集上，IGSR发现的核心项在25种子中高度一致（80%+种子包含真值项）。
5. **真实科学发现**：在RNA聚合酶II暂停研究中，IGSR发现DNA甲基化抑制暂停的假设，经5-azacytidine处理实验验证（Wilcoxon p≈10⁻⁵²），证明其从数据到可验证假设的完整能力。
6. **对LLM不敏感**：在9种不同LLM（开源/闭源）上稳定优于其他基于LLM的方法。

## 7. 优点

- **方法论亮点**：
  - **细粒度反馈**：逐项影响分数是结构化的信用分配信号，比全局标量反馈更富有信息量。
  - **解耦生成与选择**：LLM负责创造性生成（提出项），但选择由数据驱动的确定性剪枝完成，避免幻觉。
  - **高效搜索**：MCTS提供系统探索，避免局部最优。
  - **可解释性**：最终模型是 $\sum w_j \psi_j$ 的稀疏线性形式，每个组分都有明确贡献。
- **实验亮点**：
  - 包含真实湿实验验证，证明AI驱动发现的实用性。
  - 广泛的消融和鲁棒性分析（LLM敏感度、特征噪声、合成异常结构、独立生物重复等）。
  - 与多种范式方法（传统SR、黑箱、AFE、神经符号）对比，证明IGSR在不同背景下的优越性。
  - 开源代码与详细超参数，可复现。

## 8. 不足与局限

- **生成先验限制**：发现能力受限于LLM预训练分布；若所需函数形式或变换完全不在训练语料中，系统不太可能提出。
- **模型结构假设**：限定于 $f = \sum w_j \psi_j$ 形式，不能直接发现深度嵌套的非线性动态系统（如全连接神经网络）。虽然IGSR-TLO变体允许优化项内常量，但整体结构仍线性。
- **优化目标单一**：当前仅优化验证MSE，未直接融入显式稀疏性或复杂度罚项，可能无法自动平衡准确性与可解释性的帕累托前沿。
- **计算成本**：虽然比传统方法快，但仍需大量LLM API调用，对低资源场景可能昂贵。简化prompts可降低成本但性能略有下降。
- **科学假说验证仍需要人工**：最终发现需领域专家评估和实验验证，不能完全自动化。
- **潜在偏差**：LLM可能引入训练语料中的科学偏见；用户需警惕将数据驱动假设误解为已确认因果律。
