## 摘要
基于大型语言模型（LLM）的自动启发式设计（AHD）在发现高质量启发式方法方面取得了显著进展。然而，现有的基于LLM的AHD方法针对固定训练实例集优化启发式方法，在实际部署中遇到分布偏移时可能灾难性失败。我们提出了鲁棒对抗性实例搜索（RAISE），这是一个将训练分布原则性邻域内的约束最坏情况实例搜索集成到基于LLM的进化搜索循环中的框架。RAISE将鲁棒AHD视为一个约束对抗性实例搜索问题：外环通过LLM操作符进化启发式方法，而无LLM的内环通过使用带边界投影的基分布参数化，在训练实例集周围的epsilon球内高效识别困难实例。在在线装箱（OBP）、在线作业车间调度（OJSP）和在线车辆路径（OVRP）问题上的全面实验跨越五个分布族，结果表明现有的基于LLM的AHD方法在分布偏移下性能下降高达19倍，而RAISE在所有测试的分布和问题规模上始终保持强劲性能。

## Abstract
Automated Heuristic Design (AHD) with Large Language Models (LLMs) has shown remarkable progress in discovering high-quality heuristics. However, existing LLM-based AHD methods optimize heuristics for a fixed training instance set and may fail catastrophically when deployed under real-world distributional shifts. We propose Robust Adversary Instance Search (RAISE), a framework that integrates constrained worst-case instance search within a principled neighborhood of the training distribution into the LLM-based evolutionary search loop. RAISE treats robust AHD as a constrained adversarial instance search problem: the outer loop evolves heuristics via LLM operators, while an LLM-free inner loop efficiently identifies hard instances within an epsilon-ball around the training instance set using a basis distribution parameterization with boundary projection. Comprehensive experiments on Online Bin Packing (OBP), Online Job Shop Scheduling (OJSP), and Online Vehicle Routing (OVRP) across five distribution families demonstrate that existing LLM-based AHD methods degrade by up to 19 times under distribution shift, while RAISE consistently maintains strong performance across all tested distributions and problem scales

## 1. 核心问题与整体含义（研究动机和背景）

- **研究背景**：自动化启发式设计（Automated Heuristic Design, AHD）旨在自动发现高质量的启发式程序，避免人工设计。近年来，大型语言模型（LLM）被集成进进化搜索中（如EoH、ReEvo、PartEvo等），显著推动了AHD的发展。
- **核心问题**：现有LLM-based AHD方法**仅在固定的训练实例分布上优化**，产生的启发式高度特化，当部署到真实世界中遭遇**分布偏移**时，性能会急剧下降（甚至崩溃），缺乏鲁棒性。
- **现有方法的局限**：
  - 一些工作尝试用多样实例或元学习改进，但要么需要预定义分布集合，要么依赖LLM生成实例，缺乏可控性和额外成本。
  - 缺乏对分布偏移的系统性约束和稳健优化。
- **RAISE的目标**：提出一种将**约束最坏情况实例搜索**集成到LLM进化搜索循环中的框架，使启发式设计在训练分布周围的原则性邻域内具备鲁棒性，从而即使在少量训练实例下也能泛化到偏移分布。

## 2. 论文提出的方法论

### 2.1 核心思想：约束 minimax 优化
- **形式化**：将鲁棒AHD建模为一个**约束最小-最大优化问题**：
  \[
  h^* = \arg\max_{h \in \mathcal{H}} \min_{s' \in B_\varepsilon(S)} \text{eval}(h, s')
  \]
  其中 \(B_\varepsilon(S)\) 是以名义实例集 \(S\) 为中心、半径 \(\varepsilon\) 的实例级不确定集（基于归一化 ℓ1 距离）。
- 外循环（LLM驱动）：通过LLM进化操作符最大化启发式在最坏实例上的性能。
- 内循环（无LLM）：在 \(B_\varepsilon(S)\) 内发现当前最佳启发式的**最困难实例**，补充进训练集，然后重评所有启发式。

### 2.2 关键技术细节

#### 外循环：LLM驱动的启发式进化
- 维护一个启发式函数种群，使用五种LLM操作符（CREATE、E1、E2、M1、M2）生成新候选。
- 候选启发式在当前的对抗实例集上通过**均值聚合得分**进行排名（避免直接取最小值导致的不稳定），保留Top-N个体。

#### 内循环：无LLM对抗实例搜索（核心创新）
- **参数化**：将候选实例分布用一个 **18维基因向量** \(g \in [0,1]^{18}\) 表示，构造为**9种基分布**（均匀、小、大、中心、双峰、高斯、周期、泊松、峰值）的加权混合。
- **epsilon-球投影**：将得到的分布 \(p_{adv}\) 投影到距离最近名义实例 \(b\) 的 \(\varepsilon\)-球边界上，确保满足约束：
  \[
  p_{\text{proj}} = b + \frac{\varepsilon}{\max(\varepsilon, d(p_{adv}, b))} \cdot (p_{adv} - b)
  \]
- **搜索算法**：使用均匀交叉和带高斯变异的进化算法，在少量族内代数（\(G_{in}=4\)）和种群（\(P_{in}=8\)）下，找出使当前最佳启发式性能最差的可行实例。
- **频率**：每 \(\tau=5\) 个外循环代数执行一次。

### 2.3 算法流程（文字描述）
1. 初始化：从名义实例集 \(S_0\) 开始，采样初始启发式种群。
2. 外循环：每代通过LLM操作符生成新启发式，在当前实例集上评估得分，进行生存选择。
3. 每隔 \(\tau\) 代：运行内循环，得到最坏实例 \(s^*\)，将其加入实例集 \(S\)，重评所有启发式。
4. 重复直到预算耗尽，返回得分最高的启发式。
